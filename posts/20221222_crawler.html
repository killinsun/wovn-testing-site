ページとコンテンツを取得するためのツール

ページの自動追加：これにより、WOVNはターゲットサイトからWOVN.ioに新しいページを自動的に追加できます。この機能には、プロジェクトの設定からアクセスできます。

自動コンテンツ取得：この機能により、WOVNは登録されたページを定期的に自動的にスキャンして新しいコンテンツを探し、その結果を報告できます。レポートの比較で新しいコンテンツが検出された場合、このコンテンツはターゲットページにリンクされています。この機能には、プロジェクトの設定からアクセスできます。 以下は、これら2つの機能が互いにどのように連携し、プロジェクトにどのように影響するかを示す図です。

その他のAPI ほとんどのAPIはSwaggerによって処理されるため、ここですべてをグループ化しました。さまざまなタイプがありますので、詳細についてはドキュメントを参照してください。 HTMLのダウンロードとアップロードはファイル変換APIを参照していることに注意してください。

TranslationSet APIは、Foundationsチームによって処理されます。 Documentation: WOVN API Internal Notes: Translation Set API Project Overview v1 API これらのAPIは、依然として大多数のクライアントによって使用されています。ほとんどのサポートケースにはそれらが含まれます
